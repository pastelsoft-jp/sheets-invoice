# 請求書メーカー for Sheets - Feature Manifest

## Free機能
- [x] 請求書テンプレート（インボイス制度完全対応A4縦）
- [x] 消費税自動計算（10%標準/8%軽減、端数切り捨て）
- [x] 源泉徴収税額自動計算（10.21%/20.42%、ON/OFF可）
- [x] PDF出力（Google Drive保存、ダウンロードリンク）
- [x] 月3枚制限（月切り替えリセット）
- [x] オンボーディング（事業者情報・振込先入力）
- [x] リアルタイム合計計算（品目入力と同時に更新）
- [x] 登録番号バリデーション（T+13桁）

## Basic機能（¥500/月）
- [x] 無制限生成
- [x] 顧客マスタ（CRUD、最大50件）
- [x] 請求書番号自動採番（プレフィックス設定可）
- [x] テンプレートカスタマイズ（振込先、備考のデフォルト設定）

## Pro機能（¥1,500/月）
- [x] 発行履歴管理（最大100件）
- [ ] メール送信（PDF添付）※ 次バージョン
- [ ] 見積書・納品書・領収書テンプレート ※ 次バージョン
- [ ] Excel/CSV出力 ※ 次バージョン
- [ ] 複数テンプレートデザイン ※ 次バージョン
