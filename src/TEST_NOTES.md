# テスト項目

## 1. オンボーディング
- [ ] 初回起動時にオンボーディング画面が表示される
- [ ] 事業者名未入力で「はじめる」→エラーメッセージ
- [ ] 登録番号が不正形式(T+13桁以外)→エラーメッセージ
- [ ] 正常入力→設定保存→メイン画面に遷移

## 2. 請求書作成
- [ ] 品目追加ボタンで行が追加される
- [ ] 品目削除ボタンで行が削除される
- [ ] 数量・単価入力でリアルタイム合計が更新される
- [ ] 10%と8%の混在で正しく税率区分される
- [ ] 源泉徴収ON/OFFで合計が変わる
- [ ] 源泉徴収: 100万円以下→10.21%
- [ ] 源泉徴収: 100万円超→段階計算
- [ ] 取引先名未入力で生成→エラーメッセージ
- [ ] 品目なしで生成→エラーメッセージ
- [ ] 正常入力→請求書シート生成→PDF生成→結果表示

## 3. PDF
- [ ] PDFがA4縦で生成される
- [ ] PDFがGoogle Driveの「請求書PDF」フォルダに保存される
- [ ] PDFリンクが正常に開ける

## 4. 月間制限
- [ ] Freeプランで3枚目まで生成できる
- [ ] Freeプランで4枚目→エラーメッセージ
- [ ] 月が変わるとカウントリセット
- [ ] Basic/Proプランで無制限

## 5. 顧客マスタ
- [ ] Freeプランでは顧客マスタ非表示
- [ ] Basic+で顧客保存→セレクトに追加される
- [ ] 顧客選択→取引先情報が自動入力される
- [ ] 顧客50件上限でエラー

## 6. 設定
- [ ] 事業者情報の保存・反映
- [ ] 振込先の保存・反映
- [ ] 番号プレフィックスの変更
- [ ] 支払期限日数の変更
- [ ] 源泉徴収デフォルトの変更
- [ ] 全設定リセット→確認ダイアログ→リセット→リロード

## 7. Stripe決済
- [ ] アップグレードボタン→Checkout URLが開く
- [ ] ポーリングで決済完了を検知
- [ ] プラン反映（UI更新）
- [ ] Customer Portal→Stripe画面が開く

## 8. Card Service
- [ ] ホームページトリガーでメインカードが表示される
- [ ] プラン情報が正しく表示される
- [ ] サイドバー開くボタンが機能する
