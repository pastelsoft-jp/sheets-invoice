# 請求書メーカー for Sheets マーケットプレイス戦略

## 1. リスティングコピー

### アプリ名
請求書メーカー - インボイス対応 | Invoice Maker for Japan

### 1行説明（50文字以内）
インボイス制度対応の請求書をスプレッドシートから1クリックで生成

### 詳細説明

Googleスプレッドシートからインボイス制度（適格請求書等保存方式）に完全対応した請求書を作成できるアドオンです。

**こんなお悩みありませんか？**
- 請求書のインボイス制度対応が面倒
- 消費税の税率区分（10%/8%）の計算が煩雑
- 源泉徴収税額の計算を毎回手動でやっている
- freeeや弥生は高機能すぎて、請求書だけ作りたいのに...

**請求書メーカーなら、3ステップで完了**
1. 取引先と品目を入力
2. 自動で消費税・源泉徴収を計算
3. 「PDF生成」ボタンで完成

**主要機能**
- インボイス制度の必須8項目を自動表示（登録番号・税率区分・消費税額等）
- 消費税の自動計算（10%標準税率/8%軽減税率）
- 源泉徴収税額の自動計算（10.21%/20.42%）
- PDF出力（A4縦、そのまま送付可能）
- 請求書番号の自動採番
- 顧客マスタ（取引先情報の保存・呼び出し）

**料金プラン**
- 無料: 月3枚まで
- Basic ¥500/月: 無制限生成・顧客マスタ・一括生成
- Pro ¥1,500/月: メール送信・見積書/納品書/領収書・履歴管理

**データは全てGoogleの中**
外部サービスへのデータ送信はありません。請求データはお使いのGoogleスプレッドシートとGoogle Driveに保存されます。
