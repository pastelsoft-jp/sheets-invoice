# 請求書メーカー for Sheets ビジネスモデル

## 1. 料金プラン

| プラン | 月額 | 年額 | 内容 |
|--------|------|------|------|
| Free | 0 | - | 月3枚の請求書生成、基本テンプレート、消費税・源泉徴収計算 |
| Basic | 500 | 4,800 | 無制限生成、顧客マスタ50件、自動採番、一括生成、テンプレートカスタマイズ |
| Pro | 1,500 | 14,800 | メール送信、履歴管理、見積書/納品書/領収書、Excel出力、テンプレート5種 |

年額 = 月額 x 12 x 0.8（20%割引）

## 4. M&A想定

### 売却タイムライン
| 条件 | アクション |
|------|----------|
| Month 6: MRR 5,000+ | 単体売却の検討開始 |
| Month 9: MRR 10,000+ | ラッコM&Aに出品 |
| Month 12: MRR 15,000+ | 積極的に売却 |
| ポートフォリオ全体でMRR 50,000+ | ポートフォリオ売却 |

### 売却額目安
- MRR 15,000 x 24ヶ月 = ¥360,000
- MRR 30,000 x 24ヶ月 = ¥720,000
- ポートフォリオ(Calendar + Invoice + 他) MRR 50,000 x 24 = ¥1,200,000

### 売却価値を高める要素
- インボイス制度対応 = 法令遵守ツール（買い手にとって安心）
- Marketplace内の発見性が高い（「請求書」は検索頻度高）
- 維持コストゼロ
- 日本語モート（英語圏の買い手は参入不可）
