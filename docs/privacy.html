プライバシーポリシー

最終更新日: 2026年3月6日

PastelSoft（以下「当方」）が提供する「請求書メーカー for Sheets」（以下「本アドオン」）における個人情報の取扱いについて、以下のとおり定めます。

1. 収集する情報

本アドオンは以下の情報を取得します：

  • Googleアカウントのメールアドレス — ユーザー識別およびライセンス管理のため
  • 事業者情報（任意入力） — 請求書に記載する発行者名、住所、登録番号等
  • 取引先情報（任意入力） — 請求書に記載する請求先情報
  • 請求書データ — 品目、金額、税率等の請求内容

2. 情報の保存場所

すべてのユーザーデータは、ユーザー自身のGoogle Apps Script PropertiesService内に保存されます。当方のサーバーにデータが送信・保存されることはありません。

3. 第三者への提供

以下の場合を除き、個人情報を第三者に提供することはありません：

  • Stripe Inc. — 有料プランの決済処理のため、メールアドレスがStripeに送信されます。Stripeのプライバシーポリシーは https://stripe.com/jp/privacy をご確認ください。

4. Google APIの利用

本アドオンは以下のGoogle APIスコープを使用します：

  • spreadsheets — 請求書テンプレートの読み書き
  • drive.file — PDF生成・保存（本アドオンが作成したファイルのみ）
  • script.external_request — Stripe API通信
  • userinfo.email — ユーザー識別

本アドオンは、Google API サービスのユーザーデータに関するポリシー（Limited Use要件を含む）に準拠しています。

5. データの削除

サイドバーの設定タブから「全設定をリセット」を実行すると、保存されたすべてのデータを削除できます。アドオンをアンインストールした場合も、PropertiesServiceのデータは自動的に削除されます。
